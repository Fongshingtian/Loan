发展历程

  • 2018年小牛普惠在路上，再出发

  • 2017年

  • 9月

    “走过风雨，遇见彩虹”2017年小牛普惠全国大会顺利召开

  • 6月

    小牛普惠启动“爸爸很牛”公益计划，首期资助百余位脑瘫儿父亲。

  • 6月

    小牛资本董事长前往重庆，与小牛普惠重庆员工共同庆祝小牛走进新五年。

  • 5月

    小牛资本董事长彭铁前往北京、宁波“走基层”，宴请全国标杆团队，并展开一系列业务交流会。

  • 4月

    小牛普惠全国巡讲会拉开序幕，小牛金服CEO王洁凤、副总裁王芳率多名高管“走基层”，从深圳出发，奔赴福州、合肥、郑州、成都、长沙、北京等地，深入了解全国各地基层业务员的工作与生活。

  • 2月

    小牛普惠与中行、工行、友邦保险等在《晶报》第六届“百万读者金融业口碑榜”评选中，同获“百万读者金融业口碑奖”。

  • 1月

    小牛普惠携手多家同业名企共创风控协作联盟。

  • 2016年

  • 12月

    小牛普惠受深圳市关爱行动公益基金会授牌殊荣。

  • 12月

    小牛普惠受邀参加中国小额信贷国际峰会，资产端创新模式受业界关注。

  • 10月

    《小牛资本家庭金融战略暨小牛金服启动发布会》10月28日在深圳蛇口价值工厂隆重举行。

  • 3月

    3月15日第一届小牛普惠信任节举行。

  • 1月

    2016年1月小牛普惠业绩突破10亿元！

  • 2015年

  • 12月

    因在推进普惠金融发展、服务小微金融上表现突出，小牛普惠荣膺《晶报》“百万 读者金融口碑榜”--最佳小微金融服务奖。

  • 11月

    因在产品、服务和风控建设上的显著创新，小牛普惠荣登“2015中国普惠金榜”--金融业务创新奖；

  • 7月

    小牛普惠投资管理有限公司第一百家营业部于辽宁沈阳盛大开业。

  • 1月

    小牛普惠投资管理有限公司第十家营业部在广州天河东路盛大开业。

  • 2014年

  • 11月

    随着小牛普惠投资管理有限公司第一家营业部在深圳车公庙盛大开业，公司开始了全国分店网点的战略布局。

  • 7月

    小牛普惠投资管理有限公司在深圳市卓越世纪中心成立。
